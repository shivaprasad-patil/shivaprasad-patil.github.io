## Running Locally

To run this website on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/shivaprasad-patil/shivaprasad-patil.github.io.git
   cd shivaprasad-patil.github.io
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Serve locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Visit `http://localhost:4000` in your browser
